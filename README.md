# La Casa de las Cortinas | Web de Autor

Este proyecto es un ecosistema digital de alta gama diseñado bajo el estándar de **Arquitectura de la Luz**.

## 📁 Estructura del Proyecto
- `index.html`: Home principal (Colecciones, Esencia, Contacto).
- `asociados.html`: Portal Elite para profesionales (Arquitectos y Desarrolladores).
- `404.html`: Página de error personalizada.
- `index.css`: Sistema de diseño "Zero Noise" y tipografía Montserrat.
- `app.js`: Lógica del asistente virtual LUZ y efectos de revelación.
- `assets/`: Galería de renders y activos de marca.

## 🚀 Cómo Visualizar
1. Simplemente abra `index.html` en cualquier navegador web moderno (Chrome, Edge, Safari).
2. Para una experiencia óptima, se recomienda el uso de un servidor local (ej. Live Server de VS Code o `npx serve .`).

## ⚙️ Tecnologías
- HTML5 Semántico
- CSS3 (Vanilla) con Glassmorphism y Grid Layout.
- JavaScript (ES6+) con IntersectionObserver para animaciones.
